5 Tag HTML biasa digunakan

<div>

Wadah umum untuk mengelompokkan elemen. Dipakai untuk membuat struktur layout (seperti kotak-kotak ini).

<div class="container">...</div>

Contoh <div>

<a>

Digunakan untuk membuat link. Dipakai untuk navigasi antar halaman atau mengarah ke website lain.

<a href="https://link.com">Klik</a>
Klik Disini

<img>

Digunakan untuk menampilkan gambar. Dipakai untuk poster, foto profil, atau aset visual lainnya.

<img src="foto.jpg" alt="Deskripsi">

<h1>

Tag judul (Heading). Dipakai untuk memberi tahu browser bagian mana yang merupakan judul utama teks.

<h1>Judul Laporan</h1>

Judul Laporan

<p>

Tag paragraf. Dipakai untuk menuliskan teks narasi atau penjelasan panjang dalam sebuah dokumen.

Kendala Yang dialami

Selama proses pembelajaran Frontend Development, kendala utama yang dihadapi terletak pada pemahaman mendalam mengenai Layouting dan Positioning.

#1

Kesulitan mengatur tata letak elemen <div> menggunakan properti CSS konvensional yang sering kali tidak presisi.

#2

Tantangan dalam memosisikan Gambar Responsif agar tetap proporsional di berbagai ukuran layar.

#3

Teknik perataan elemen ke tengah (Centering Strategy) baik secara vertikal maupun horizontal yang memerlukan logika Box Model & Flexbox

Bagian yang paling mudah/sulit

Sisi_Mudah: Visual

Berbekal pengalaman sebagai Graphic Designer, proses menentukan komposisi warna, pemilihan tipografi, dan pembuatan aset visual terasa jauh lebih intuitif.

"Mata desainer membantu saya mempercepat proses prototyping di Figma sebelum masuk ke tahap coding."

Sisi_Sulit: Logic

Tantangan terbesar muncul saat harus merealisasikan Dashboard Kompleks. Mengatur posisi elemen yang saling bergantung satu sama lain sering kali menjadi kendala teknis yang nyata.
